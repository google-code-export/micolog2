Alltuts是来自Site5的Wordpress主题，几个老外在主题设计上很厉害，做出来的东西非常大气，一款看起来很简洁的主题，在细节上却毫不含糊。这款Alltuts在wopus主题站是四星半的评价，适合喜欢简洁和重视用户体验的用户。

言归正传，移植的Alltuts在相似度上可达到99%，Sidebar和Comments都是看着Site5上的演示页面手动敲出来的，界面和功能绝对和原版一模一样，完全可伪装成WP。原版Alltuts使用了不少Jquery和WP插件，这里都剔除了，改用CSS来实现效果和相应功能，以提高运行速度。

附加功能：在原版基础上，增加了带用户头像的首页“最新评论”，底部Foot部分借用“公告栏”，以实现原版“About This Site”模块，支持HTML显示，随时随地可以更换显示内容。评论改用AJAX提交（虽然我不知道啥是AJAX），添加了回复邮件通知功能。页面也加入了“评论”部分，不需要可在后台发布文章时取消评论。已加入高亮标记，支持代码高亮显示，同时支持0.7以上版本自带的谷歌访问统计。右侧边栏中已预留广告区域，广告图片都在images文件夹中，可链接或替换图片，别忘了图片下面还有一个文字链接广告，一并替换。如您不投放广告，可删除广告标签，或替换成“公告栏”，来增加博客功能。

本主题支持代码高亮，谷歌统计等功能，只要在博客后台的“插件”中激活并进行相应设置即可使用。

注意此主题不支持IE6，其它IE7+和FF、Chrome、Opera等常用浏览器均兼容，如主题有问题给我留言，附上原版演示地址：

您需要修改的部分：

原版中带的是Twitter微博，那啥你知道的，带这东西不仅显示不出来，而且还会拖慢网页打开速度，所以这里改为了Sina微博，http://open.t.sina.com.cn/widgets 新浪微博客的Widget地址，登录自己的账号后，点击微博秀，设置一个256px宽度的微博秀挂件，并勾选掉标题栏、边框等，只留下微博内容模块，之后把生成的iframe代码复制下来，替换掉主题 sidebar.html中的iframe部分，完成！你也可以把它换成QQ微博，或在底部放QQ微博，两全其美。

底部“博客日志”，删除Base.html中我键入的废话，留下{{blog.blognotice}}变量即可。

由于是移植老外的主题，所以首页RSS链接部分，都是Facebook等老外喜欢的热门社会化分享站点，需要换成自己的链接。当然，也可以换成国内的热门社会化分享站点。

文章页部分，同样是twitter、Facebook等老外喜欢的社会化分享网站，需要换成自己的链接，或更改为国内的社会化分享站点。

首页导航上，左边的是目录导航，不需要目录中文章数显示的，删除base.html中的({{cate.count}})就行。

使用此主题时，请在后台控制面板中设置验证码方式为“图片”，不仅防垃圾评论效果更好，本主题“点击显示”验证码图片的方式，还可避免验证码过期的问题出现。
